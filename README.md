# チンチロ - 丁半サイコロ

器に3つのサイコロを振って出目と役を表示するスマホ向けWebアプリです。

## 遊び方

1. 「振る」ボタンをタップする
2. 器が揺れ、サイコロが回転するアニメーションのあと出目が決まる
3. 役（ピンゾロ・ゾロ・シゴロ・ヒフミ・通常役・役なし）が表示される
4. 「役一覧」で各役の説明を確認できる

## 役の説明

| 役 | 出目 | 意味 |
|---|---|---|
| ピンゾロ | 1・1・1 | 最高役 |
| ゾロ | 同じ目3つ | アラシ（2〜6のゾロ） |
| シゴロ | 4・5・6 | 強役 |
| ヒフミ | 1・2・3 | 即負け |
| 通常役 | 同じ目2つ | 目で強弱を比較 |
| 役なし | 上記以外 | 負け |

## 起動方法

ブラウザで `index.html` を開くか、ローカルサーバーで配信してください。

```bash
# 例: Python 3
cd chinchiro
python3 -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

スマホで使う場合は、同じWi-Fi内でPCのIPアドレスにアクセスするか、GitHub Pages などにデプロイしてURLで開けます。

## ファイル構成

- `index.html` - 画面構造
- `styles.css` - レイアウト・器とサイコロの見た目・アニメーション
- `script.js` - 振る処理・役判定・UI操作
